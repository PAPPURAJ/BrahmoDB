# BrahmoDB

BrahmoDB is an Oracle-inspired relational database engine built from scratch in Java.

The goal of this project is to understand database internals step by step, including:

- SQL parsing
- Table storage
- Page-based storage
- Buffer pool
- Indexing
- Transactions
- Locking
- Query execution
- Client-server communication

## Tech Stack

- Java 21
- Maven
